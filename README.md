# GUESSING-A-RANDOM-NUMBER-GAME:
## DESCRIPTION:
This is a simple pytjhon number guessing game. the program generates a random number between 1 and 100, and the user has to guess the correct number. it provides hints whether the guess should be higher or lower and displays the total number of attempts.

## FEATURES:
- Random number generation
- Higher/Lower hints
- Attempt counter
- User-friendly interface

## TECHNOLOGIES USED:
- Python 3
- random module

## HOW TO RUN:
1. Clone the repository.
2. Open the project in VS code.
3. Run the python file:
   ```bash
   python guessing_game.py
   ```

## EXAMPLE OUTPUT:
```
Guess the number : 45 
Wrong! Guess Higher
Guess the number : 70 
Wrong! Guess Lower 
Guess the number : 62
Congratulations! Correct Guess 
Attempts: 3
```
